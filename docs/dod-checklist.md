# Definition Of Done Checklist

## Global
- Feature code implemented.
- Typecheck and lint pass.
- Relevant tests pass.
- API error paths return structured errors.
- Auth protection enforced.

## Backend module
- Input validation present.
- DB writes are guarded and consistent.
- Lookup/aggregation behavior matches spec.
- Soft-delete or publish-state rules respected.

## Frontend module
- Mobile tap targets >= 44px.
- No horizontal overflow.
- Loading and error states present.
- Primary user action visible and reachable quickly.
